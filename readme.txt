------------------ Terceira Entrega ------------------


Ficheiros pertencentes a entrega: server.py, client.py, queries.py e aitd.db

Comandos disponiveis no cliente: ADD ALUNO, ADD DISCIPLINA, ADD TURMA, ADD <id_aluno> <id_turma>, SHOW/REMOVE ALUNO, SHOW/REMOVE TURMA, SHOW/REMOVE ALL ALUNOS|TURMAS|DISCIPLINAS, SHOW/REMOVE <id_alunos> <id_turma>, SHOW/REMOVE ALL ALUNOS TURMA, SHOW/REMOVE ALL ALUNOS DISCIPLINA, SHOW/REMOVE ALL TUMAS.

Guia de utilização: correr com um ficheiro de base de dados presente no mesmo directorio do server. se nao, incluir um ficheiro sql com o codigo sql do enunciado. iniciar o client e executar os comandos acima descritos.
